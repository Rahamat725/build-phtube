Discuss the scope of var, let, and const

Var

i. Variables declared with 'var' is function-scoped. When a variable is declared with 'var' inside function, it will not access outside function.

ii. Variables declared with 'var' are hoisted to the top of their containing function or global scope.

iii. Variables declared with 'var' can reassigned and redelared within same scope.

Let

i. Variables declared with 'let' are Block-scoped. When it declared inside any block like(if, loop, funtion etc.), it will not access outside the scope.

ii. Variables declared with 'let' are hoisted, but not initialized. if we try to use before declaration it will show error!.

iii. Variables declared with 'let' can reassigned witin same block but can't redelare.

Const

i. Variables declared with 'const' are Block-scoped it will not access out side block-scope.

ii. Variables declared with 'const' are hoisted but not initialized like 'let'. to use 'const' before declaration will show error.

iii. constant variable can't be changed after initialization.

Tell us the use cases of null and undefined

undefined

i. When a variable declared but not initialized, it has default value 'undefined'.

ii. When a function does not return value it implicitly return 'undefined'.

iii. When we try to access any object property and if does't exist it return 'undefined'.

null

i. when we try to get the object value and if the value does not exist it return 'null' value.

ii. null can be use to comparisons to check if the variable or object property are empty or not.
